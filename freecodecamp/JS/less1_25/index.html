Оператор неравенства ( !=) противоположен оператору равенства. то означает не равно и возвращает falseто, что вернулось бы к равенству, trueи наоборот . Как и оператор равенства, оператор неравенства преобразует типы данных значений при сравнении.

Примеры
1 != 2 // true
1 != "1" // false
1 != '1' // false
1 != true // false
0 != false // false
