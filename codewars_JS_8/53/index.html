Описание:

Создайте функцию с именем _if, которая принимает 3 аргумента: логическое значение bool и 2 функции (которые не принимают никаких параметров): func1 и func2
Когда bool соответствует истине, следует вызвать func1, в противном случае вызвать func2.


Пример: 

_if(true, function(){console.log("True")}, function(){console.log("false")})

// Logs 'True' to the console.
