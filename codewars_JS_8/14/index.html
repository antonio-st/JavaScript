Описание: 
 Дан массив из трех неотрицательных целых чисел a, b и c, определить, 
            образуют ли они пифагорову тройку.

        Пифагорова тройка образуется, если:
        с2 = а2 + b2
        
        где c — наибольшее значение a, b, c.
        
        Например: a = 3, b = 4, c = 5 образуют пифагорову тройку, 
        потому что 52 = 32 + 42
        Возвращаемые значения
        
             1, если a, b и c образуют пифагорову тройку
             0, если a, b и c не образуют пифагорову тройку
             Для JavaScript: вернуть true или false
